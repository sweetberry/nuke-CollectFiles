nuke-CollectFiles
====

## Install

- Open your `.nuke` folder

- copy ```collectFiles.py``` file
into your `.nuke` folder.

- Edit your `menu.py` file, insert row:
```python
import collectFiles
```

- restart nuke.  

- select menu File >> collectFiles

- enjoy!
